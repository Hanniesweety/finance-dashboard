# Finance Dashboard UI

This project is a modern Finance Dashboard interface built using React and Chart.js. It allows users to track financial activities, analyze spending patterns, and explore transaction data through interactive charts and tables.

## Features
- Financial summary cards showing balance, income, and expenses
- Interactive charts for spending breakdown and monthly financial comparison
- Transactions table with search and filtering
- Role-based UI simulation (Viewer / Admin)
- Insights section highlighting key spending patterns
- Clean and responsive dashboard layout

## Tech Stack
- React
- JavaScript
- Chart.js
- CSS

## Purpose
The goal of this project is to demonstrate frontend development skills including UI design, component structure, data visualization, and state management.
